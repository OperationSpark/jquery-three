EXERCISE

Build an e-commerce site (like Amazon) that lists items for sale. The items should be stored as an array of objects and then displayed onto the page. The user should be able to fill out form inputs to search and sort products by certain properties (price, category, keyword, etc). Challenge yourself to use as many different underscore functions as possible, even if it seems a bit contrived.
